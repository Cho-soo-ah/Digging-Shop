# Digging Shop

💡 쇼핑몰 관리자 페이지


### 사용 기술
---

- Mongo DB, express, multer, socket.io (로컬환경)
- React (v6)
- MUI
- axios
- recoil
- formik, yup
- nivo

### 기능
---

- 메인 페이지
    - 대쉬보드
    - nivo 차트 라이브러리 사용
- 상품 관리
    - 카드형 리스트
    - 수정, 위시리스트 저장, 카트에 담기 기능(로컬)
- 매출 관리
    - 매출 정보 리스트
    - 엑셀로 다운로드 버튼
- 배송 관리
    - 배송 정보 리스트
    - 엑셀로 다운로드 버튼
- 회원 관리
    - 회원 정보 리스트
    - 엑셀로 다운로드 버튼
    - 회원 추가 페이지(로컬)
- 이벤트, 공지사항, 1:1 문의
    - params 받아서 상세페이지 라우팅
- 위시리스트
    - 좋아요 버튼을 누른 상품 리스트(로컬)
- 마이페이지
    - 개인 정보 수정, 저장(로컬)
    - 주문내역 리스트
- 장바구니
    - 카트에 담기 버튼을 누른 상품 리스트(로컬)
    - 결재금액 계산기 기능(미구현)
- 로그인/회원가입
    - 유효성 검사
    - 회원가입 시 회원 관리 페이지에 업데이트
